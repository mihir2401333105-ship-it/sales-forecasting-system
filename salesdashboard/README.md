# Sales Forecast AI Dashboard

A modern, high-performance dashboard for visualizing AI-powered sales forecasts.

## Features
- Interactive Line & Bar charts (Recharts)
- Glassmorphism & Neon Design
- CSV Data Upload support
- Real-time prediction analysis
- One-click dataset export

## Tech Stack
- React (Vite)
- Recharts
- Tailwind CSS
- FastAPI (Backend)
- Scikit-learn (ML)

## Setup
1. Standard Node installations: `npm install`
2. Run development server: `npm run dev`
3. Backend: Run `uvicorn api_with_cors:app --reload`
